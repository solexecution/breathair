# Tactical Breathing

A mobile-first box breathing app with visual, audio, and haptic feedback. Used by Navy SEALs and military personnel for stress relief and focus.

![Tactical Breathing](https://img.shields.io/badge/version-1.0.0-amber)

## Features

- **Visual feedback** - Animated breathing circle that expands/contracts
- **Audio feedback** - Soft tones for each phase transition
- **Haptic feedback** - Vibration patterns on mobile devices
- **History tracking** - Year-long visualization of completed sessions
- **Multiple patterns** - Box (4-4-4-4), Relaxing (4-7-8), Energizing, Calming
- **PWA ready** - Install on mobile home screen

## Controls

- **Tap** the circle to start/pause
- **Pull down** to reset
- **Complete 4+ cycles** to mark day as done
